Trang kỹ năng

Kỹ năng kỹ thuật

HTML

HTML là xương sống của mọi trang web, phối hợp cùng CSS (định dạng) và JavaScript (tương tác) để tạo nên trải nghiệm web hoàn chỉnh.

Mức độ: Intermediate

CSS

CSS (Cascading Style Sheets) là ngôn ngữ định kiểu được sử dụng để mô tả cách trình bày và giao diện của các tài liệu viết bằng ngôn ngữ đánh dấu như HTML hoặc XML.

Mức độ: Intermediate

JS

JavaScript (JS) là ngôn ngữ lập trình kịch bản (scripting language) phổ biến nhất thế giới, được thiết kế để tạo ra các tương tác động, mang lại trải nghiệm sống động cho website. Nó chạy chủ yếu trên trình duyệt (client-side), biến HTML/CSS tĩnh thành giao diện trực quan và hoạt động nhúng trực tiếp vào file HTML.

Mức độ: Intermediate

PHP

PHP (viết tắt của "PHP: Hypertext Preprocessor") là một ngôn ngữ lập trình kịch bản mã nguồn mở, chạy trên máy chủ (server-side) phổ biến nhất hiện nay, được thiết kế đặc biệt để phát triển ứng dụng web động. Nó dễ nhúng vào HTML, dễ học, cú pháp đơn giản và có thể kết nối với hầu hết cơ sở dữ liệu như MySQL.

Mức độ: Intermediate

Bootstrap

Bootstrap là framework front-end mã nguồn mở miễn phí phổ biến nhất, giúp thiết kế website responsive (tương thích thiết bị di động) nhanh chóng bằng HTML, CSS, và JavaScript. Nó cung cấp sẵn các thành phần giao diện như nút, bảng, biểu mẫu, và hệ thống lưới.

Mức độ: Intermediate

React

React (hay ReactJS) là một thư viện JavaScript mã nguồn mở được Meta (Facebook) phát triển, dùng để xây dựng giao diện người dùng (UI) tương tác, linh hoạt và hiệu suất cao cho web. Với kiến trúc dựa trên Component và công nghệ Virtual DOM, React giúp tạo ra các ứng dụng đơn trang (SPA) mượt mà và tối ưu hóa việc render.

Mức độ: Intermediate
